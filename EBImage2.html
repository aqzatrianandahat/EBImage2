setwd("D:/Gambar")

Panggil Package

Jika sudah menginstal EBImage maka langsung kita panggil packagenya

library(EBImage)

Kita panggil gambar yang kita inginkan

setwd("D:/Gambar")
library(EBImage)

Histogram

hist(Gambar)

Menampilkan Dengan method=“browser”

Method ini akan menampilkan gambar yang bisa di diperbesar(Zoom In) dan diperkecil(Zoom Out)

display(Gambar, method="browser")

Menambahkan Teks Pada Gambar

Pada method ini kita dapat memberikan label/teks pada gambar

display(Gambar, method="raster")
text(x = 5, y = 350, label = "Saya Aqza TAHAT", adj = c(0,1), col = "black", cex = 1)

Brightness

Kita bisa mengubah tingkat kecerahan gambar dengan tanda -/+ sesuai keinginan

Gambar1 <- Gambar + 0.2
Gambar2 <- Gambar - 0.2
par(mfrow= c(1,2))
plot(Gambar1)
plot(Gambar2)

Adjusting Contrast

Kontras bisa diubah dengan menurunkan atau meningkatkan mengunakan tanda "*"

Gambar3 <- Gambar * 0.5
Gambar4 <- Gambar * 2
par(mfrow= c(1,2))
plot(Gambar3)
plot(Gambar4)

Gamma Correction

Tingkat gamma juga bisa diubah menggunakan tanda “^”

Gambar5 <- Gambar ^ 2
Gambar6 <- Gambar ^ 0.7
par(mfrow= c(1,2))
plot(Gambar5)
plot(Gambar6)

Cropping

Crop/memangkas gambar sesuai keinginan tetapi tidak boleh melewati ukuran asli dari gambar karena akan error

display(Gambar[100:300, 2:150,])

Spatial Transformation

Untuk setting rotasinya sesuai value yang ada di Imagetr <- translate(rotate(Gambar, 45), c(50, 0))

Gambartr <- translate(rotate(Gambar, 45), c(50, 0))
display(Gambartr)

Filtering

Dengan filtering kita dapat menghasilkan gambar seperti berikut

setwd("D:/Gambar")
library(EBImage)
fLow <- makeBrush(21, shape= 'disc', step=FALSE)^2
fLow <- fLow/sum(fLow)
Gambar.fLow <- filter2(Gambar, fLow)
display(Gambar.fLow)

fHigh <- matrix(1, nc = 3, nr = 3)
fHigh[2, 2] <- -8
Gambar.fHigh <- filter2(Gambar, fHigh)
display(Gambar.fHigh)

aa<- medianFilter(Gambar.fHigh,1.1)
par(mfrow= c(1,2))
plot(aa)
plot(Gambar.fHigh)

Gambar <- readImage("4x6.jpg")
medFltr <- medianFilter(Gambar, 1.1)
display(medFltr)
